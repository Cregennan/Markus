# Проект Markus

Поздравляю! Вы создали проект **Markus**!

## Возможности

На данный момент, конвертер поддерживает:
- Создание списков
- Добавление изображений
- Добавление выделений (жирный шрифт итд)
- Добавление гиперссылок
- Переносы строк
- Горизонтальная полоса
